# 5 分钟掌握 Markdown

## 标题
```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## 文本格式
```
**粗体文本**
*斜体文本*
***粗体斜体***
~~删除线~~
`行内代码`
```

## 链接和图片
```
[链接文本](https://example.com)
[带标题链接](https://example.com "悬停标题")
![替代文本](image.jpg)
![带标题图片](image.jpg "图片标题")
```

## 列表
```
无序列表：
- 项目 1
- 项目 2
  - 嵌套项目

有序列表：
1. 第一项
2. 第二项
3. 第三项
```

## 引用块
```
> 这是一个引用块
> 多行内容
>> 嵌套引用
```

## 代码块
````
```javascript
function hello() {
  console.log("Hello, world!");
}
```
````

## 表格
```
| 表头 1 | 表头 2 | 表头 3 |
|----------|----------|----------|
| 行 1    | 数据     | 数据     |
| 行 2    | 数据     | 数据     |
```

## 分隔线
```
---
或
***
或
___
```

## 任务列表
```
- [x] 已完成任务
- [ ] 未完成任务
- [ ] 另一个任务
```

## 转义字符
```
使用反斜杠转义：\* \_ \# \`
```

## 换行
```
在行尾加两个空格  
可以创建换行。

或使用空行创建新段落。
```
