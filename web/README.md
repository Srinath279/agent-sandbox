# Agent-Sandbox Website

This is the website for user guide and API reference of Agent-Sandbox https://agent-sandbox.github.io, use GitHub Pages to host the static files.

## Build and Deploy
Auto push web dir to `gent-sandbox/agent-sandbox@main` when updated, and deploy to GitHub Pages.

### Workflow Config Ref: 
1. https://github.com/marketplace/actions/push-directory-to-another-repository-directory
1. GitHub Token Settings: https://github.com/settings/tokens?type=beta
